# Media drop zone

Put images / GIFs here and they deploy with the site (served at /dev/media/...).

Folders:
- immersed/  -> Immersed card (Meta full-body avatar networking, Genies footage, AOI)
- arthur/    -> Arthur card (networked physics GIFs, network LOD)
- fignet/    -> FigNet/Entangle/FnVoice (spatial grid, architecture diagrams)
- games/     -> games & tools (PokerVR, Epic Battles, etc.)

Rules of thumb:
- Images / GIFs up to a few MB: drop them right here, I'll embed them.
- Large videos (.mp4): prefer a YouTube or Vimeo link instead (keeps the repo light) —
  just paste the link in chat and I'll embed the player.
- Name files simply, e.g. immersed/meta-avatar.gif, arthur/networked-physics.gif.

## Captions
For each file, add a one-line caption. Easiest: paste them in chat, OR add them to
captions.md in the same folder like:
  meta-avatar.gif = Full-body Meta avatar synced across the network in real time.
